Multimodal Neuro Imaging Techniques

Advanced neural imaging platform that combines multiple imaging modalities including MRI, fMRI, DTI, and PET scans. Our AIML-powered solutions provide comprehensive brain structure and function analysis for research and clinical applications.

Dataset-1: Simultaneous EEG and fMRI signals during sleep from humans.

This dataset includes EEG-fMRI recordings from 33 healthy participants at Pennsylvania State University. Each session consists of anatomical imaging, resting-state fMRI (before and after a visual-motor adaptation task), and multiple sleep sessions. EEG was recorded using a 32-channel MR-compatible system at 5000 Hz, synchronized with fMRI triggers. Sleep stages (wakefulness, NREM1-3) were manually scored and stored in TSV files. fMRI data were collected on a 3T Siemens Prisma scanner using MPRAGE for anatomical images and EPI for BOLD signals. The dataset provides multimodal neural signals to study resting-state and sleep-related brain activity.

How to Predict

  1. Step 1: Upload Your Dataset

    User selects and uploads a CSV file containing EEG (Alpha, Beta, Gamma, Delta) and fMRI (Mean BOLD, Variance BOLD) features.

  2. Step 2: Data validation

    Backend (Flask) validates the input file to match the trained model's format.

  3. Step 3: Read CSV file

    Flask reads the CSV, extracts required columns (Alpha, Beta, Gamma, Delta, Mean BOLD, Variance BOLD). Check for missing/invalid values and scale values using saved scaler.pkl file

  4. Step 4: Model Inference

    The model is loaded [saved as .h5 file] and preprocessed features are sent to the deployed model for data analysis and prediction.

  5. Step 5: Task Decoding

    Convert the model's label-encoded prediction (0/1) into a human-readable task label using the task encoding file.

  6. Step 6: Return Prediction

    Send the predicted task label and any relevant visualizations back to the frontend.

  7. Step 7: Display Prediction and Visualisation

    Display the predicted task label and visualizations (EEG/fMRI plots) on the UI.
