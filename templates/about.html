About This Project

Project Overview

The EEG-fMRI Fusion Analysis project is an advanced web application designed to analyze multimodal neuroimaging data for brain state prediction. By leveraging both electroencephalography (EEG) and functional magnetic resonance imaging (fMRI) data, our application provides more comprehensive insights into brain activity patterns.

Our fusion model has been trained on extensive datasets to recognize three distinct brain states:

  • Rest: Default mode network activation, minimal sensory processing
  • Eyes Open: Alert wakefulness, visual processing active
  • Eyes Closed: Increased alpha rhythm, reduced visual cortex activity

How It Works

The application processes CSV files containing specific EEG and fMRI features:

EEG Features:
  • Delta Power (0.5-4 Hz)
  • Theta Power (4-8 Hz)
  • Alpha Power (8-13 Hz)
  • Beta Power (13-30 Hz)
  • Gamma Power (30-100 Hz)
  • Spectral Entropy
  • Mean Coherence
fMRI Features:
  • BOLD Mean
  • BOLD Variance
  • ALFF Mean (Amplitude of Low-Frequency Fluctuations)
  • ALFF Variance

These features are processed through our fusion model, which outputs predicted brain states for each data point in your dataset.

Data Visualization

Our application provides several visualizations to help you interpret the results:

  • Feature Distributions: Displays the distribution of EEG and fMRI features in your dataset
  • Confusion Matrix: Shows the model's prediction accuracy compared to ground truth labels (if provided)
  • Feature Correlations: Illustrates relationships between different EEG and fMRI features
  • Prediction Distributions: Visualizes the distribution of prediction probabilities for each class

Data Requirements

To use this application, prepare your data in CSV format with the following columns:

Delta Power,Theta Power,Alpha Power,Beta Power,Gamma Power,Spectral Entropy,Mean Coherence,BOLD Mean,BOLD Variance,ALFF Mean,ALFF Variance[,State]

The State column is optional and should contain one of the following values if present: "Rest", "Eyes Open", or "Eyes Closed". Including this column allows the application to evaluate prediction accuracy.

Technical Details

The application is built using the following technologies:

  • Backend: Flask (Python)
  • Data Processing: NumPy, Pandas
  • Visualization: Matplotlib, Seaborn
  • Frontend: HTML5, CSS3, JavaScript, Bootstrap
  • Animation: Particles.js

The model architecture integrates EEG and fMRI features using a neural network fusion approach to leverage the complementary information from both modalities.
